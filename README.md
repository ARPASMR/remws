# eiffel
REMWS eiffel code

This repository contains remws public eiffel code. Enjoy!

# Service daemon
Il demone del servizio si chiama remwsgwyd. 
Per farlo partire
```
service remwsgwyd start
```
## Opzioni
-p port (default 9090)

-l log level

-syslog usa syslog per le notifiche (default: false)

-h help
