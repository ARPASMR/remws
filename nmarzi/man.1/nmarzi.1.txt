NAME
----
nmarzi - nmarzi simulator


SYNOPSIS
--------
*nmarzi*


DESCRIPTION
-----------
The nmarzi eiffel application clones the old nmarzi windows application.


REMARKS
-------
`nmarzi` reads 'nmarzi.preferences.xml' file in the home user folder.


EXAMPLE
-------
`nmarzi`::
  Does it.
